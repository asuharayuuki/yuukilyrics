# yuukilyrics 🎵

A modern, cross-platform LRC lyric editor designed for high-precision karaoke timing, with general compatibility with **RhythmicaLyrics**.

------

## Build

### Prerequisites

- Install the latest stable version of the **Flutter SDK**.
- Windows/Linux only: make sure `ffmpeg` is added to your system environment variables.

### Getting Started

1. Clone this project to your local machine.

2. Install all Flutter dependencies in the project root directory:

   ```bash
   flutter pub get
   ```

3. Start the development environment:

   - **Windows**:

     ```bash
     flutter run -d windows
     ```
